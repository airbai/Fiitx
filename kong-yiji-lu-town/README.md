# 🎭 孔乙己 · 鲁镇游览

> 灵感来自 [Messenger](https://messenger.abeto.co/) — 一款由 abeto 制作的清新小星球 WebGL 小游戏

## 概述

基于 **Three.js** 构建的沉浸式 3D 漫游体验。以鲁迅先生名篇《孔乙己》中的鲁镇为舞台，玩家操控孔乙己漫步六个经典场景，在低多边形风格的江南古镇中重温那段「大约孔乙己的确死了」的悲凉故事。

## Messenger 风格致敬

| 特征 | Messenger | 本作 |
|------|-----------|------|
| 🌍 **世界** | 小星球（球形曲面） | 弧形景观（small‑planet curvature） |
| 🎨 **色调** | 青绿 + 白色，明亮清新 | 暖白 + 青绿 + 大地色 + 彩色花朵 |
| 🏔️ **造型** | 低多边形（low‑poly） | 低多边形建筑、树木、人物 |

| 🔄 **加载** | SVG 路径动画 | SVG 路径序列动画 |
| 🖱️ **交互** | 点击 + 轨道控制 | 点击行走 + 点击光点 + 轨道控制 |
| 🎵 **音景** | — | 五声音阶古风环境音（Web Audio） |

## 特色功能

### 🫘 茴香豆收集
场景中散落着 **12 颗茴香豆**（金色发光漂浮物），操控孔乙己走近即可收集。全部收集后触发彩蛋：「多乎哉？不多也。」

### 👥 NPC 人物
- **掌柜** — 站在曲尺柜台后
- **短衣帮** — 咸亨酒店旁的酒客
- **孩子们** — 街巷中玩耍的孩童
- **路人** — 偶遇的读书人

### 🎵 环境音景
首次点击自动启动 **Web Audio API** 生成的五声音阶古风氛围音，音量随呼吸缓慢调制。

### 🚶 行走动画
孔乙己行走时手臂自然摆动，身体微微上下起伏，方向平滑转向。

### 🌈 天空渐变
Messenger 风格的上蓝下白渐变穹顶，柔和的地平线光晕。

## 操作

| 操作 | 效果 |
|------|------|
| 👆 **点击地面** | 孔乙己走过去 |
| ✨ **点击光点** | 触发该地点的故事叙述 |
| 🫘 **走近金色豆子** | 自动收集茴香豆 |
| 🎯 **右侧圆点** | 快速跳转到各地点 |
| 🔄 **拖拽画面** | 旋转/拉近拉远视角 |
| 🗺️ **右下小地图** | 实时显示位置和已访问地点 |
| 📍 **左上豆子计数** | 显示已收集的茴香豆数量 |

## 技术栈

- [Three.js](https://threejs.org/) — 3D 渲染引擎
- CSS2DRenderer — 标签系统
- OrbitControls — 相机控制
- Web Audio API — 环境音效
- 纯前端，无构建工具，打开即玩

## 运行

直接用浏览器打开 `index.html`：

```bash
open index.html
```

或通过本地服务器：

```bash
npx serve .
```
